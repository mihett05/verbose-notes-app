## Running locally
Install deps
```shell script
npm install
# or
yarn install
```

#### For development with hot reload
```shell script
npm run dev
# or
yarn dev
```

Open [localhost:3000](http://localhost:3000/)

#### For production
```shell script
npm run build
npm start
# or
yarn build
yarn start
```

Open [localhost:3000](http://localhost:3000/)
